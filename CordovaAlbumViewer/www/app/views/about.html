About the AlbumViewer Sample

Cordova

This sample application demonstrates using Cordova to port a small Web application to run as a native mobile application. The sample is set up to run on iOS, Android and Windows Phone 8.x.

Local Data

This application runs all local data on device, with initially loaded data from static files that are then maintained in localstorage. This was done to have a self-contained app without external dependencies, but the app actually contains code that can connect to an external service.

Angular and BootStrap

This example uses AngularJs as the application framework, and modified Bootstrap styling to provide this Cordova mobile app interface.
